## Local Usage

### Build and Install Packages
```bash
$ git clone https://github.com/ilyakmet/wsRetranslator
```

```bash
$ cd wsRetranslator
```

```bash
$ bash install.sh
```

### Run Retranslator
```bash
$ retranslator
```
